# cinema-management-system（影票优选）
### 项目简介

影票优选是一个实现了电影的管理以及用户购票的项目

### 项目技术栈

Vue2 + ElementUI + Axios + Vuex + VueRouter + WebSocket

### node版本
node 16

### 拉取项目后执行安装依赖
```
npm install
```

### 运行项目
```
npm run serve
```

### 构建项目
```
npm run build
```
